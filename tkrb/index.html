Additionally, there are two JSON files with mappings of sword ids to names and data that can either be saved locally (names and data), or imported into a Javascript module as import sword_names from './sword_data/name_long.json' with {type: "json"}; (names) or import swords from './sword_data/swords.json' with {type: "json"}; (data). The data version contains information like rarity, sword school, and type, but may not be as up-to-date.
