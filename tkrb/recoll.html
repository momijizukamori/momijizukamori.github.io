TKRB Wiki Recollection Formatter

Takes the browser JSON response for recollections and formats it it to be pasted into the TKRB wiki. Recollection data is available by opening the dev tools network tab (Ctrl + Shift + e) before triggering the recollection, and looking for a request to https://w011.touken-ranbu.jp/scene/reproduce - the response to that is the JSON to be pasted into the input box.
